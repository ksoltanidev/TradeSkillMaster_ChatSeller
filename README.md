# TradeSkillMaster_ChatSeller

A TSM module for chat-based selling automation on Ascension WoW.

## Requirements

- World of Warcraft 3.3.5 (WotLK) - Ascension WoW
- TradeSkillMaster

## Installation

1. Download or clone this repository
2. Copy the `TradeSkillMaster_ChatSeller` folder to your AddOns directory:
   `C:\Program Files\Ascension Launcher\resources\client\Interface\AddOns\`
3. Restart WoW or type `/reload`

## Usage

- Open TSM with `/tsm`
- Click the ChatSeller icon in the sidebar
- Or use `/tsm chatseller` to open directly

## Development

Use `deploy.bat` to copy the addon to your game folder (requires admin rights).
